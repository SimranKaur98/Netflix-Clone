# Netflix-Clone
Netflix-inspired web application that replicates the core UI and browsing experience of Netflix, allowing users to explore movies and TV shows in a clean, responsive interface.

# Tech Stack used: React (Vite, npm), CSS, TMDB API, JavaScript (ES6+), Express, Firebase DB

# Why this project is useful
It demonstrates real-world frontend skills such as component-based architecture, API integration, state management, and responsive designâ€”exactly what production apps need.

# What it does (Core Functionality)

  Displays movies and TV shows by category
  Fetches real-time data from an external movie API
  Shows posters, titles, and previews
  Responsive layout for desktop and mobile
  Smooth UI interactions similar to Netflix
  Seamless video embedding

# What matters most (Core Insight)
This project focuses on scalable UI design + API-driven data flow, which is the backbone of modern streaming platforms.
